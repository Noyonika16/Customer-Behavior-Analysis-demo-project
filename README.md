# 🛍️ Customer Shopping Behavior Analysis (Learning Project)

This repository contains my implementation of an end-to-end Data Analytics project completed for learning and practice purposes. The project helped me gain hands-on experience with a typical data analytics workflow involving Python, SQL, and Power BI.

> **Disclaimer:** This is **not an original project idea**. It is a practice implementation based on the excellent portfolio project created by **Amlan Mohanty**. I recreated the analysis myself to strengthen my practical skills in data analytics tools and workflows.

---

## 📌 Project Overview

The objective of this project was to understand and practice the complete analytics pipeline used in industry, including:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Loading data into a SQL database
- Writing SQL queries to answer business questions
- Building an interactive Power BI dashboard

## 📊 Workflow

### 1. Data Preparation (Python)

- Imported the dataset
- Performed data exploration
- Handled missing values
- Checked duplicates
- Feature engineering
- Data cleaning and preprocessing

### 2. SQL Analysis

- Connected Python with PostgreSQL
- Loaded the cleaned dataset into SQL
- Wrote SQL queries to answer business questions such as:
  - Revenue analysis
  - Customer segmentation
  - Purchase behavior
  - Discount analysis
  - Category-wise sales
  - Loyalty analysis

### 3. Power BI Dashboard

Created an interactive dashboard to visualize:

- Revenue metrics
- Customer demographics
- Category performance
- Purchase trends
- Discount impact
- Customer behavior insights
